XMLTV

Overview

Freevo's TV guide is populated using data obtained by XMLTV. XMLTV is a set of utilities (scripts) to manage your TV viewing. They work with TV listings stored in the XMLTV format, which is based on XML. The idea is to separate out the backend (getting the listings, e.g. tv_grab_uk_rt) from the frontend (displaying them for the user, e.g. freevo's TV menu), and to implement useful operations like picking out your favourite programmes as filters that read and write XML documents.

At present (24Jan2005) there are backends grabbing TV listings for Canada, the USA, the UK, Germany, Austria, Finland, Spain, Italy, the Netherlands, Hungary, Denmark, Japan, Sweden, France, Norway, and Romania.

Once you have successfully configured XMLTV you are close to a working TV setup for Freevo. Please refer to the configuration section for TV_Channels to complete your setup.
