AnalogTvConfig

Important Freevo uses other programs in order to provide the user with the content they requested (dvd, mp3, photo, tv). In the case of TV viewing freevo uses either mplayer or tvtime. Before you continue with your setup please ensure that you are able to view TV using these apps on their own.

Overview

Freevo can use either TvTime or Mplayer to view TV. Analogue configuration is done from your preferences files freevo.conf and local_conf.py. As previously mentioned you need to have XMLTV configured at this stage, please refer to the relevant docs for this. The infomation below will give details on the specific configuration necessary.

  • Configure freevo: Configuring Freevo for analogue TV

  • Card Types: Types of analogue TV cards

  • Multi Cards: Use of multiple TV cards

  • Frequencies: Use of channel frequencies (alternative config, not requirement)

  • IVTV Cards: Information on configuring Freevo for use with the IVTV driver for PVR-1/2/350 cards
