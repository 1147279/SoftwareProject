FreevoAptDebian

Freevo Apt for debian

A Debian apt source is available and includes all the dependencies you'll need.

You should add this to your /etc/apt/sources.list file: 

deb http://freevo.sourceforge.net/debian unstable main
deb ftp://ftp.nerim.net/debian-marillat/  unstable main

(The second source provides mplayer and other multimedia packages)

To fetch all the dependencies you'll require to run Freevo simply run 

apt-get update
apt-get install freevo

If you are using a debian 'stable' version such as the version called 'woody', your regular /etc/apt/sources.list will be not be configured by default in such a way that allows apt-get to find all the dependencies. In particular, things such as python2.3 are not included in the 'woody' release. If you don't fear an update of your installation you could update your /etc/apt/sources.list by changing 'stable' and 'woody' into 'sarge' and rerun the two commands listed above.

If you are installing afresh, it would be wise to choose a version of Debian like 'sarge' so that your apt sources will enable apt-get to find all the dependencies for the freevo package. By the time you read this, 'sarge' may already be lifted from 'testing' to 'stable' (May 2005).

Ubuntu: See FreevoAptUbuntu

Some hints for installing freevo on the Debian-based Ubuntu distribution can be found under FreevoAptUbuntu.
