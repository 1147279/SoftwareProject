6.2. Downloading TV Logos

Freevo comes with a helper application that downloads the logos specified in the TV.xml file created by XMLTV. Freevo then uses these images in its TV guide, which makes it easier to browse the various channels. 

      [user@host]$ ./freevo execute helpers/makelogos.py
