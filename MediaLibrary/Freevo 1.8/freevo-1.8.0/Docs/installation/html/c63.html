Chapter 2. Prerequisites

2.1. General

It is assumed that you have successfully installed Linux on your soon-to-be-a-PVR machine and that you have a good knowledge of Linux. Furthermore, it is assumed that you have previously recompiled a kernel and know how that process works.

If you are new to Linux or you have never recompiled your kernel you might want to check out The Linux Kernel HOWTO and the linux.org documentation section.
