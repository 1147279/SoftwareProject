5.4. Changing Icons

It's pretty easy to change the icons. There are a few ways to change it. You can change the icon path in your skins XML file, or your can copy new images over the current skin's image files. Being a lazy person and not wanting to break anything I decided to copy my new image files over the old ones. If you are using the Blue Round skin the icon files are located in skins/icons/AquaFusion and are aptly named "tv.png" and the like. I downloaded the images from http://halfempty.com/bright/pics/freevo, which look great.
