Chapter 8. Freevo User Manual

This section is dedicated to actually using your Freevo box. After you have your box all set up and working properly you will want to actually use it. If you want, print this section out and put it on your coffee table for quick reference.

8.1. General

8.1.1. Navigation

To navigate the menus use the arrow keys for navigation and the ENTER key to select an item. To go back to the previous menu hit the ESC key.

8.1.2. Help

To view the on screen help menu hit the "h" key.
