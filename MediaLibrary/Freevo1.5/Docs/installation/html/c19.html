Chapter 1. Introduction

1.1. Introduction

When I first decided to pursue creating my own Linux based PVR I started with a desire and a computer. Unfortunately, there wasn't much else on the Internet in the way of documentation. The Freevo team has done a great job in creating software, which hasn't left much time for good documentation, which is why I decided to create this document. Through this document I hope to enable other people to build their own Freevo based PVR / Media boxes.

If I have missed anything or completely messed up something please email me with suggestions, bugfixes and corrections. This document is not, by any means, a complete document.
