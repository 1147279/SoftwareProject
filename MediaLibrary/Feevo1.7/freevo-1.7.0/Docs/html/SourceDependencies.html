SourceDependencies

Overview

  • Freevo has a large number of dependencies. If you are using a distro that already has Freevo pre-packaged it is recommended to use them. For the rest of us and those that just like getting dirty here is the core list:

Python

  • Python 2.3 (Python 2.4 has unresolved issues) http://www.python.org/2.3.5/

  • Pygame http://www.pygame.org/

  • mmpython http://sourceforge.net/projects/mmpython/

  • PyXML http://pyxml.sourceforge.net/

  • Egenix (python-mx-base) http://www.egenix.com/files/python/eGenix-mx-Extensions.html#Download-mxBASE

  • PIL (Python Image Library) http://www.pythonware.com/products/pil/

  • Python Twisted http://twistedmatrix.com/products/download

  • Numeric http://numeric.scipy.org/

Libraries

  • libexif http://sourceforge.net/projects/libexif

  • SDL http://www.libsdl.org/download-1.2.php

  • SDL_Mixer http://www.libsdl.org/projects/SDL_mixer/

  • SDL_ttf http://www.libsdl.org/projects/SDL_ttf/

  • SDL_image http://www.libsdl.org/projects/SDL_image/

  • Freetype 2 http://sourceforge.net/project/showfiles.php?group_id=3157&package_id=3121

  • smpeg 4.4 http://www.lokigames.com/development/smpeg.php3

  • jpgtran http://www.ijg.org/files/jpegsrc.v6b.tar.gz

  • aumix http://sourceforge.net/project/showfiles.php?group_id=405&package_id=445

  • lsdvd http://sourceforge.net/projects/acidrip/

  • libdvdread http://www.dtek.chalmers.se/groups/dvd/downloads.shtml

  • libdvdcss http://developers.videolan.org/libdvdcss/

Core Applications

  • tvtime http://tvtime.sourceforge.net/

  • Mplayer http://www.mplayerhq.hu/homepage/design7/dload.html

  • Xine-0.9.22 or newer http://xinehq.de/index.php/releases

Perl Modules (optional, for xmltv only)

  • LWP

  • XML::Twig

  • XML::Parser

  • XML::Writer

  • Date::Manip

    The simplest way to get install the perl modules is to use "cpan <module name>".

Depending on what features you want beyond Core there are extra deps. Future updates to this page will attempt to make a complete list of deps and which apps depend on which to make building easier.
