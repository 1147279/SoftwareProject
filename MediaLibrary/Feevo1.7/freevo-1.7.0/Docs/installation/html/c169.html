Chapter 3. Installing / Configuring Freevo

3.1. Installing Freevo

It wouldn't be a PVR / Media box without Freevo! I downloaded the binary distribution from the official Freevo website, which should include everything you need to start up Freevo. Once you have untar'd Freevo do the following: 

    [user@host]$ ./configure
    [user@host]$ cp freevo_config.py local_config.py
    [user@host]$ ln -s /path/to/freevo_source /usr/local/freevo

That's pretty much it. After you have done the above you will need to start configuring Freevo's config files.
