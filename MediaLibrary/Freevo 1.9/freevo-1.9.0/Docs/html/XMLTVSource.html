The source code for XMLTV can be downloaded here:

XMLTV: [WWW] http://sourceforge.net/project/showfiles.php?group_id=39046

To install run:

./configure

make

make install

Note: XMLTV has a number of dependencies, for more complete instructions on installing XMLTV please refer to their documentation.
