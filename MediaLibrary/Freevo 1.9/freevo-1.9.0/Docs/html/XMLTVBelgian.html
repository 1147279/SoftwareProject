XMLTVBelgian

Belgian(Dutch) listings

The old xml_tv_be.py script has been removed from the Freevo CVS repository. A new grabber was developped to replace it. The new tv grabber is part of the PyTVGrab package and can be found at the following adress : http://pytvgrab.sourceforge.net

Usage is the same as the Perl XMLTV scripts. More documentation can be found on the PyTVGrab Site

  • download the guide for the next two days : 

# tv_grab_be_tvb -n2 -o /tmp/TV.xml

  • download the guide for the next n days, example 7 days : 

# tv_grab_be_tvb -n7 -o /tmp/TV.xml
