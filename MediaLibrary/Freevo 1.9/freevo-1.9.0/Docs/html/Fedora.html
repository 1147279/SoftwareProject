Fedora

Fedora

ALERT: Redhat 9 support is deprecated for freevo version 1.5 due to a lack of proper unicode support in python 2.2.x. Please upgrade to at minimum Fedora Core 2/3. Proper Fedora Core 4 support will require Freevo > 1.5.3 (pending).

ALERT: The yum repository paths have been changed to cater for differences between Fedora Core 2/3 and Fedora Core 4. Please update your yum repository information for freevo.repo based on the example below (notably, the addition of $releasever to the baseurl path).

Freevo yum repository

Edit /etc/( yum.conf | yum.repos.d/freevo.conf ) | (fc3/fc4: /etc/yum.repos.d/freevo.repo) 

[freevo]
name=Freevo RPM Repository for Fedora Core
baseurl=http://freevo.sf.net/fedora/$releasever
gpgkey=http://freevo.sourceforge.net/fedora/tcwan_freevo_key.asc
enabled=1
gpgcheck=1

Dependencies

Dag Wieers

For Fedora Core 2 / 3, you can use Dag to satisfy the dependency packages.

/etc/( yum.conf | yum.repos.d/dag.conf ) | (fc3: /etc/yum.repos.d/dag.repo) 

[dag]
name=Dag RPM Repository for Fedora Core
baseurl=http://apt.sw.be/fedora/$releasever/en/$basearch/dag
gpgkey=http://dag.wieers.com/packages/RPM-GPG-KEY.dag.txt
enabled=1
gpgcheck=1

Dries

RPM packages for Fedora Core 4 are maintained by Dries.

/etc/yum.repos.d/dries.repo 

[dries]
name=Extra Fedora rpms dries - $releasever - $basearch
baseurl=http://ftp.belnet.be/packages/dries.ulyssis.org/fedora/linux/$releasever/$basearch/dries/RPMS/
gpgkey=http://dries.ulyssis.org/rpm/RPM-GPG-KEY.dries.txt
enabled=1
gpgcheck=1

Freshrpms

/etc/( yum.conf | yum.repos.d/fresh.conf ) | (fc3/fc4: /etc/yum.repos.d/fresh.repo) 

[freshrpms]
name=Fedora Linux $releasever - $basearch - freshrpms
baseurl=http://ayo.freshrpms.net/fedora/linux/$releasever/$basearch/freshrpms
gpgkey=http://freshrpms.net/RPM-GPG-KEY-freshrpms
enabled=1
gpgcheck=1

[updates]
name=Fedora Linux $releasever - $basearch - updates
baseurl=http://ayo.freshrpms.net/fedora/linux/$releasever/$basearch/updates
gpgkey=http://freshrpms.net/RPM-GPG-KEY-freshrpms
enabled=1
gpgcheck=1
