8.2. TV Guide

8.2.1. Changing Stations

There are two ways in which you can change stations. If you are currently watching a channel and would like to go up or down a channel type "c" or "v". Another option is to press "ESC" to return to the TV Guide and then use the arrow keys to navigate to the correct channel. Hit "ENTER" once you have found the desired station.

8.2.2. Recording Shows

To record a show you need to go to the TV Guide. Once you are in the TV Guide browse the listings using the arrow keys. After you have highlighted the show you wish to record hit "F6" to bring up the recording menu.

Please note that shows may not be recorded properly unless you have set Freevo up to check for recordings via Cron. See Freevo Recording Daemon for more information.
